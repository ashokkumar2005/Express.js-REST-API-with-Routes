
# Node.js Express Backend - Routes & Middleware

This project is a simple backend API built using **Node.js** and **Express.js**.  
It demonstrates how to structure a backend application using **routes, controllers, middleware, and utility functions**.

## Features

- REST API using Express
- Clean folder structure
- Routes and controllers separation
- Middleware usage
- Utility functions
- JSON request handling

## Tech Stack

- Node.js
- Express.js
- JavaScript
- Nodemon
